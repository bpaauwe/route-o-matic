Route-O-Matic Documentation

When the app starts, it displays the main naviation menu along the top. Below that is some text about the app. And below that is the main interface to the routes.

Navigation

Top/Left is "Route-O-Matic". Clicking this will always bring you back to the main screen.

Next is "About". This takes you to a page with a bit of information about the app.

Then comes "Documentation". This is what you're reading now.

Last is "Login". Having a valid account might eventually provide some additional capabilities. Currently, it does nothing.

Main interface to the routes

First is the route selector drop down. Clicking this will show all the routes in the system by title. Selecting one will allow you to view/print the route.

Print format is another drop down. This determines the output format you get when you select Printable. The available formats are:

  1. default - A casual format that looks nice when printed.
  2. Spreadsheet - Formatted like a spreadsheet with each element in a cell with borders.
  3. Rallymaster - Some routes may have additional notes embedded that are for the event leader/rallymaster. This is formatted like the default format but includes those extra notes.

The Printable button will generate a print ready page. You should be able to send this page to your printer directly. You may have to adjust some print settings like headers/footers. Use your browser's back button to get back to the main interface.

The Edit button currently does nothing. Additional features may make use of it in the future.

Below all the controls is a text box. When you select a route, the route description will show up in this box.

Final notes

The public version you are using was adapted from a private version that was used to enter all of the available routes.

* Many of these routes were created in the early 2000's and are now 20 to 26 years old.
* Some of the businesses used for start/finish locations may no longer exist.
* Some of the roads may have been re-aligned or are no longer suitable for car club events.
* The routes were entered based on the material provided by the original author of the event.
* Make sure you check out the route before using it for an event!
